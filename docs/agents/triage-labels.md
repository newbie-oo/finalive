# Triage labels

| Role            | Label             |
| --------------- | ----------------- |
| Needs triage    | `needs-triage`    |
| Needs info      | `needs-info`      |
| Ready for agent | `ready-for-agent` |
| Ready for human | `ready-for-human` |
| Won't fix       | `wontfix`         |

These strings are used as directory names under `.scratch/<feature>/` for local-markdown issue tracking.
